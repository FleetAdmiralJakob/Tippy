# Tippy - A Simple Tip Calculator
A tip calculator (for Android)!

![Tippy](Images/README%20Image%20v2.1.png)

****Author:** [Jakob Rössner](https://www.roessner.tech)**

Features:
- Pure Kotlin
- Single screen
- No libraries
- Language support
